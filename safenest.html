SafeNest – Women Safety Web App

Problem Statement

In emergency situations, especially for women, there is often a lack of quick, accessible, and discreet safety tools. Existing solutions are either slow, require multiple steps, or fail to provide real-time evidence collection and location tracking.

Solution

SafeNest is a browser-based safety application designed to provide instant emergency assistance. It enables users to trigger an SOS alert, share their real-time location, and record audio/video evidence discreetly using built-in browser APIs.

Key Features

  • 🚨 One-click SOS alert system
  • 📍 Real-time GPS location tracking
  • 🎥 Hidden audio and video recording
  • ⚡ Fast response without app installation

Tech Stack

HTML CSS JavaScript Geolocation API MediaRecorder API

Challenges Faced

Handling real-time location accuracy and ensuring seamless media recording across browsers were major challenges. Managing permissions for camera, microphone, and location while maintaining a smooth user experience required careful implementation.

Future Improvements

  • Integration with emergency contacts and SMS alerts
  • AI-based threat detection
  • Mobile app version for wider accessibility
