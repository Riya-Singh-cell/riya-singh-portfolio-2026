PrepGenie – AI Study Assistant

Problem Statement

Students often struggle to manage multiple academic tasks such as generating notes, preparing for exams, and planning study schedules. Existing tools are either static or focus on a single feature, making learning inefficient and fragmented.

Solution

PrepGenie is an AI-powered full-stack study assistant that integrates AI-based note generation with cloud storage. It enables users to generate structured notes using LLMs and download them anytime through a scalable backend system.

Key Features

  • 📚 AI-generated study notes
  • ☁️ Cloud storage using AWS S3
  • ⬇️ Downloadable notes (Text File)
  • ⚡ Real-time API interaction (FastAPI)
  • 🎯 Scalable full-stack architecture

System Architecture

Frontend (React) → FastAPI Backend → Groq LLM → File Generation → AWS S3 → Public URL → Download

Tech Stack

React Tailwind CSS Python FastAPI Groq API AWS S3 boto3

Key Implementation

Built a REST API pipeline to generate notes using LLMs and store them in AWS S3. Implemented cloud storage with unique file naming and public URL generation, enabling persistent and scalable access to generated study material.

Challenges Faced

Integrating frontend, backend, AI, and cloud into a single workflow required careful system design. Handling CORS issues, managing dependencies, and configuring AWS S3 permissions were key challenges.

Future Improvements

  • Secure downloads using pre-signed URLs
  • User authentication system
  • Dashboard for saved notes
  • Mobile app version
