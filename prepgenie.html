PrepGenie – AI Study Assistant

Problem Statement

Students often struggle to manage multiple academic tasks such as generating notes, preparing for exams, and planning study schedules. Existing tools are either static or focus on a single feature, making learning inefficient and fragmented.

Solution

PrepGenie is an AI-powered study assistant that integrates multiple learning tools into one platform. It helps students generate notes, create mock tests, and build personalized study plans using intelligent AI systems.

Key Features

  • 📚 AI-generated notes with explanations
  • 📝 Customizable mock test generation
  • 📅 Personalized study planner
  • ⚡ Real-time AI responses using LLMs
  • 🎯 All-in-one learning assistant platform

Tech Stack

React Tailwind CSS Python FastAPI Groq API LLMs

Challenges Faced

Integrating multiple AI features (notes, tests, planner) into a single seamless workflow required careful system design. Managing prompt engineering for different tasks and ensuring consistent output quality across features was a major challenge.

Future Improvements

  • User authentication and progress tracking
  • Adaptive learning based on performance
  • Analytics dashboard for study insights
  • Mobile app version
